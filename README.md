# Resume-Builder

How to run (quick start)
Backend

Create folder backend, paste files there.

cd backend

npm install

Copy .env.example to .env and fill MONGO_URI and JWT_SECRET.

npm run dev (requires nodemon) or npm start.

Frontend

Create folder frontend, paste files there.

cd frontend

npm install
